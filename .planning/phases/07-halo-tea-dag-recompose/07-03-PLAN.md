# Phase 7 Plan 03: Tool Lifecycle + ToolManagerAgent (No MCP Dependency)

<objective>
实现 TEA 语义下的工具生命周期治理基线：新增 ToolLifecycle 服务与 ToolManagerAgent，并在 `chat_with_tools` 打点生命周期事件。默认开关关闭，不改变现有执行路径。
</objective>

<threat_model>
- Threat: runtime bootstrap regression due to new role branch. Mitigation: add only `role=="tool_manager"` minimal branch; keep existing role resolution unchanged.
- Threat: tool lifecycle instrumentation changes business behavior. Mitigation: gate by `ENABLE_TOOL_LIFECYCLE=false` by default and keep return payload unchanged.
- Threat: event emission side effects. Mitigation: best-effort emission, swallow lifecycle telemetry errors.
</threat_model>
