# Phase 7 Plan 02: MCTS Shadow Strategy Trace (No Runtime Behavior Change)

<objective>
实现 HALO 风格 MCTS Shadow 选择：在 5 个候选策略中执行 UCB 评分并输出策略轨迹，仅写入 trace，不改变 DAG 执行行为。
</objective>

<threat_model>
- Threat: strategy selector accidentally changes execution path. Mitigation: shadow-only integration into `decomposition_trace.strategy_trace`; scheduler behavior untouched.
- Threat: nondeterministic trace weakens regression. Mitigation: deterministic seed from task input (`title|mode|depth|target_words`).
- Threat: config rollout risks. Mitigation: feature flag `ENABLE_MCTS_SHADOW=false` by default.
</threat_model>
