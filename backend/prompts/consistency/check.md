## 检查维度
1. **风格统一性**：各章节的语言风格、人称、时态是否一致
2. **逻辑连贯性**：章节间论述是否有矛盾或跳跃
3. **重复内容**：是否有跨章节的重复描述
4. **术语一致性**：同一概念在不同章节中的称呼是否统一
5. **衔接自然性**：章节间过渡是否自然

## 输出格式
返回严格JSON格式：

```json
    }}
  ]
}}
```

## 判定规则
- 无 high 级别问题 → pass = true
- 有 high 级别问题 → pass = false
- issues 列表只包含需要修改的问题，不报告正常的内容
